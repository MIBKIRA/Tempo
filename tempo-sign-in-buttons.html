Design note

Engineered control, not gradient decoration. The color behind each button is two flat plates, not a blurred glow — hover lifts the button off them, a press flattens it flush, like a real toggle switch bottoming out. Corners are cut on one diagonal, not rounded on all four. Focus draws two viewfinder brackets instead of a browser outline. Loading is a four-bar meter, not a spinner.
